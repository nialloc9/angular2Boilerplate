# angular2_base_template
This is the template set up for angular2 including notes from the documentation on what everything does. This build was created using typescript
which is compiled into javascript. To use this follow the instructions below.

1. Download and install nodejs. https://nodejs.org/en/download/

2. Download this repoitory.

3. Install the libraries and dependencies. Do this by going to the root directory and using the following command: npm install

4. Start the typescript compiling and and tun the lite server which has auto-refreshing: npm start

5. If successful browser will open and show first 'loading..' then 'My First Angular 2 App'
